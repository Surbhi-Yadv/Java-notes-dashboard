Unit 2 - Variables and Data Types

Variables store values and data types tell what kind of values can be stored.

  • Meaning of variable - A variable in Java is a container used to store data values during program execution. Each variable has a data type, which defines the type of value it can hold, such as integer, float, or string. In Java, variables must be declared before use and can be initialized with a value. There are mainly three types of variables: local variables, which are declared inside methods and used only within that method; instance variables, which are declared inside a class but outside methods and belong to objects; and static variables, which are declared using the static keyword and are shared among all objects of a class. Variables help in storing and manipulating data, making programs dynamic and flexible.
  • Primitive data types -Bigava has 8 primitive data types: byte Size: 1 byte Used to store small integers Example: byte b = 10; short Size: 2 bytes Stores medium-range integers Example: short s = 200; int Size: 4 bytes Most commonly used integer type Example: int x = 1000; long Size: 8 bytes Stores large integers Example: long l = 100000L; float Size: 4 bytes Stores decimal numbers Example: float f = 10.5f; double Size: 8 bytes More precise decimal values Example: double d = 20.99; char Size: 2 bytes Stores single character Example: char c = 'A'; boolean Size: 1 bit (conceptually) Stores true or false Example: boolean b = true;
  • Non-primitive data types Types of Non-Primitive Data Types String Used to store text Example: String name = "Surbhi"; Array Stores multiple values of same type Example: int[] arr = {1, 2, 3}; Class Blueprint to create objects Example: class Student { int marks; } Object Instance of a class Example: Student s = new Student(); Interface Contains abstract methods (used for abstraction)
  • Type casting -BigType casting means converting one data type into another data type. ✨ Types of Type Casting 1. Implicit Casting (Widening) Done automatically by Java Converts smaller data type → larger data type 2. Explicit Casting (Narrowing) Done manually by programmer Converts larger data type → smaller data type
