Unit 5 - OOP Concepts

OOP stands for Object-Oriented Programming. It is based on class and object.

  • Class and object --A class in Java is a blueprint or template used to create objects. It defines properties (variables) and behaviors (methods) of an object. An object is an instance of a class that represents a real-world entity and can access the properties and methods defined in the class.
  • Constructor - A constructor in Java is a special method used to initialize objects. It has the same name as the class and is automatically called when an object is created. Constructors are used to assign initial values to variables and can be of different types like default and parameterized constructors.
  • Inheritance - Inheritance is a feature in Java that allows one class to acquire the properties and methods of another class. It helps in code reusability and reduces duplication. The class that inherits is called the child class, and the class being inherited from is called the parent class.
  • Polymorphism - Polymorphism means “many forms.” It allows one method or object to behave differently in different situations. In Java, it is achieved through method overloading (same method name with different parameters) and method overriding (same method in parent and child class with different implementation).
  • Encapsulation and abstraction - Encapsulation is the process of wrapping data and methods into a single unit (class) and restricting direct access using private variables and public methods (getters and setters). Abstraction means hiding complex details and showing only essential features to the user, making the program simpler and easier to use.
