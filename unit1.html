Unit 1 - Introduction to Java

Java is a high-level, object-oriented programming language. It is platform independent because Java runs on JVM.Java is a popular high-level programming language developed by James Gosling in 1995 at Sun Microsystems. It is widely used for building applications like web, mobile, and desktop software. Java follows the principle of “Write Once, Run Anywhere”, which means a Java program can run on any system that has a Java Virtual Machine (JVM). It is known for its simplicity, security, and object-oriented features, which make programming easier and more efficient. Today, Java is one of the most widely used languages in the world and is important for software developmen

  • History of Java : Java was developed by James Gosling and his team at Sun Microsystems in 1991. Initially, it was created for electronic devices like set-top boxes and was called “Oak” (named after an oak tree near Gosling’s office). Later, in 1995, the name was changed to Java and officially released. The main goal of Java was to create a language that is simple, secure, and platform-independent (Write Once, Run Anywhere). In 2009, Oracle Corporation acquired Sun Microsystems, and since then Oracle manages Java and continues to improve it with new versions.
  • Features of Java 1 - Simple Java is easy to learn and use because its syntax is similar to C/C++ but simpler. 2 -Object-Oriented Java follows OOP concepts like class, object, inheritance, and polymorphism. 3 Platform Independent Java follows Write Once, Run Anywhere (WORA) — it runs on any system with JVM. 4 -Secure Java provides strong security features like bytecode verification and no pointer access. 5 -Robust It is reliable because of strong memory management and exception handling. 6 -Multithreaded Java supports multiple tasks at the same time (multithreading).
  • Applications of Java 1- Web Applications Java is used to create dynamic websites and web apps (like login systems, portals). 2 -Mobile Applications Java is widely used for Android app development using Android Studio. 3 -Desktop Applications Java is used to build desktop software like calculator, notepad, etc. 4 -Enterprise Applications Java is used in large systems like banking, ERP, and management software. 5 =Cloud Applications Java helps in developing cloud-based services and applications. 6 -Big Data Applications Tools like Apache Hadoop are built using Java.
