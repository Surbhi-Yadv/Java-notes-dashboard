Unit 4 - Control Statements

Control statements manage the flow of execution in a Java program.

  • if statement - The if statement in Java is used to check a condition. If the condition is true, the block of code inside the if statement is executed. It helps in decision making by allowing the program to take action only when a specific condition is satisfied.
  • if-else and nested if - The if-else statement is used when there are two possible outcomes. If the condition is true, one block executes; otherwise, the else block runs. A nested if means using an if statement inside another if statement to check multiple conditions step by step.
  • switch statement - The switch statement is used to select one block of code from many options. It works with different cases, and the program executes the matching case based on the given value. It is useful when there are multiple choices instead of using many if-else statements.
  • for, while, do-while loops- Loops in Java are used to repeat a block of code multiple times. The for loop is used when the number of iterations is known. The while loop runs as long as a condition is true. The do-while loop executes the code at least once before checking the condition.
  • break and continue- The break statement is used to stop the execution of a loop or switch statement immediately. The continue statement is used to skip the current iteration of a loop and move to the next iteration. These statements help in controlling the flow of loops.
